# 동작구 · 서울특별시

> 동작구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-dongjak.github.io/**

동작구는 한강 남안에 자리한 서울의 자치구로, 조선 시대 한양과 삼남을 잇던 나루 '동작진'에서 이름을 물려받은 고을입니다. 노량진 학원가에 매일 수만 명의 수험생이 모여드는 대한민국 대표 교육 상권이자, 새벽마다 경매 소리가 울리는 노량진수산시장이 있는 활력의 도시입니다. 순국선열이 잠든 국립서울현충원과 단종 복위에 목숨을 바친 여섯 충신을 기린 사육신공원이 한강변을 따라 이어져 역사의 무게를 더합니다. 중앙대·숭실대·총신대가 언덕에 둥지를 튼 대학가와, 상도동·흑석동의 오래된 주택가가 어우러져 젊음과 정주가 공존합니다. 정조가 아버지 능행길에 장승을 세웠다는 장승배기에는 2025년 신청사와 종합행정타운이 들어서 구의 새로운 중심으로 떠올랐습니다.
